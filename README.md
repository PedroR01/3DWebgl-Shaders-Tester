# 3DWebgl-Shaders-Tester

Web app to test differents 3D geometries and shaders options using Webgl with Threejs library in real time.

## Aditional steps to work with shaders and initialize a project like this from scratch

1. Create a vanilla Vite template.
2. Install threejs lib.
3. Install GLSL pluggins for your IDE (VsCode in my case).
4. Install GLSL pluggin for vite compatibility with command 'npm install vite-plugin-slsl --save-dev'.
5. Check vite.config.js content.
